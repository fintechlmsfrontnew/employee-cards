# Quick Upload Guide - Best Method

## Step 1: GitHub Desktop Install (2 minutes)

1. Download: https://desktop.github.com/
2. Install karo
3. GitHub account se login karo

## Step 2: Repository Add (1 minute)

1. GitHub Desktop open karo
2. **"File" > "Add Local Repository"**
3. `C:\Users\User\CascadeProjects\employee-cards` select karo
4. **"Add Repository"** click karo

## Step 3: Publish to GitHub (1 minute)

1. Repository name check karo: `employee-cards`
2. **"Publish Repository"** button click karo
3. GitHub account select karo: `fintechlmsfrontnew`
4. **"Publish Repository"** click karo

## Step 4: GitHub Pages Enable (2 minutes)

1. GitHub repository open karo: https://github.com/fintechlmsfrontnew/employee-cards
2. **"Settings"** tab click karo
3. **"Pages"** section mein jao
4. **"Source"**: "Deploy from a branch" select karo
5. **Branch**: `main`
6. **Folder**: `/root`
7. **"Save"** button click karo

## Step 5: Test Website (5 minutes wait)

Wait 5 minutes, then test:
- https://fintechlmsfrontnew.github.io/employee-cards/directory.html
- https://fintechlmsfrontnew.github.io/employee-cards/employees/employee1.html

## Total Time: 10-15 minutes

## Benefits:
- No authentication issues
- Simple GUI interface
- Automatic HTTPS setup
- Easy for future updates
